Lưu ý quan trọng về nguồn ảnh/model

  • Việc tự sinh pattern cần đọc trực tiếp pixel ảnh trong trình duyệt (CORS). Google Drive đôi khi chặn việc này.
  • Nếu "Test train pattern" báo lỗi CORS, hãy đổi ảnh marker sang nguồn cho phép CORS: Imgur (link ảnh trực tiếp .jpg/.png), GitHub raw, ImgBB, Firebase Storage...
  • Model 3D cũng cần nguồn cho phép tải cross-origin để hiển thị được trong camera AR ở trang AR.html.

Tên AR

Đây là ảnh marker người dùng sẽ cần quét bằng camera ở trang AR để xem mô hình 3D hiện lên.

Cách hoạt động

  1. Bạn dán link ảnh marker + link model 3D đã upload lên bên thứ 3.
  2. Hệ thống tự đọc ảnh, chia lưới 16×16, lấy mẫu màu từng ô → sinh ra dữ liệu pattern (giống file .patt) và lưu thẳng vào Firestore (không cần host file .patt riêng).
  3. Trang AR công khai (ar.html) sẽ tải toàn bộ pattern này về và dùng AR.js để nhận diện nhiều marker cùng lúc.
